Title:                   Sr Pentaho Developers

Duration:            6+months

Location:            San Diego, CA

Job Description:-

  • 4-6yrs of Experience in IT industry
  • Must have done atleast one end to end implementation using Pentaho Suite & associated tools/technologies
  • Must be familiar with configuration and usage of Pentaho
  • Must be familiar with ETL & Dashboarding on Pentaho
  • Expert on coding and reviewing code written in Pentaho (SQL-like + custom scripting)
  • Atleast 2yrs of experience in Pentaho Suite is a must
  • Must be capable of leading and guiding colleagues and team members in implementation of technical requirements specified in design documents
  • Preferable to have experience in creating low level design documents detailing the technical architecture and build on Pentaho
  • Must have exposure to working / managing / leading a LIVE deployment and maintenance implementation on Pentaho

 

FTE/Permanent Opening For TIBCO BE Consultants 

Work Location: Herndon, VA 

Experience: 8 - 15 Years 

 

Role, responsibilities, and skill sets:

 ·         Working experience in creating Concepts, Simple Events, Complex Events, DB concept, writing the rules and configuring the backing store in TIBCO BE.

·         Experience in designing the Business State Model, Decision Table using TIBCO Business Events.

·         Expertise in Application Architecture and integration of Web based/n-tier architecture applications.

·         Knowledge of EAI messaging standards like WebSphere MQ and JMS messaging, using both point-to-point and pub/sub messaging models.

·         Working experience in design, development and implementation of enterprise level framework.

·         Working experience in EDI, XML, Flat file transactions and communication.

·         Experience with relational databases including DB2, SQL Server and Oracle.
